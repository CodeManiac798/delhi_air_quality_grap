## Refreshing these

If the dashboard's layout or content changes, re-export and overwrite the
files above using the same filenames — the root README already links to
these exact paths, so no other edit is needed. In Power BI Desktop, open
`Delhi_GRAP_Analytics_Platform.pbip`, go to each page, and use
**File → Export → Export to PDF** (then convert each page to PNG) or a screen
capture at a consistent window size. Recommended: 1920×1080 or the report's
native 1280×720, PNG format, light theme (unless intentionally showcasing
dark mode as a separate set).
