# Dashboard Screenshots

Screenshot exports of each report page, referenced by the root README's
[Dashboard Preview](../../README.md#dashboard-preview) section.

## How to generate

In Power BI Desktop, open `Delhi_GRAP_Analytics_Platform.pbip`, go to each
page, and use **File → Export → Export to PDF** (then convert each page to
PNG) or a screen capture at a consistent window size. Save into this folder
using the filenames below — the root README already links to these exact
paths, so no other edit is needed once the files are in place.

| Page | Filename |
|---|---|
| Executive Overview | `page1_executive_overview.png` |
| Air Quality Analysis | `page2_air_quality_analysis.png` |
| GRAP Intervention Analysis | `page3_grap_intervention_analysis.png` |
| Weather Context Analysis | `page4_weather_context_analysis.png` |
| Methodology & About | `page5_methodology_about.png` |

Recommended: 1920×1080 or the report's native 1280×720, PNG format, light
theme (unless intentionally showcasing dark mode as a separate set).
